
# Customer Churn Prediction

## 📌 Project Overview

This project predicts which customers are likely to stop using a telecom service using machine learning models.

## 🔍 Workflow

* Data preprocessing and EDA performed in Google Colab
* Multiple models trained and evaluated:

  * Logistic Regression
  * Random Forest
  * XGBoost

## 📊 Results

| Model               | Accuracy | Recall (Churn) | ROC-AUC |
| ------------------- | -------- | -------------- | ------- |
| Logistic Regression | 0.75     | 0.13           | -       |
| Random Forest       | 0.78     | 0.42           | 0.819   |
| XGBoost             | 0.77     | 0.49           | 0.816   |

## 🛠 Tools Used

* Python
* Pandas
* Scikit-learn
* XGBoost
* Google Colab
